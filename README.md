# 学习笔记站

一个基于 VitePress 构建的个人学习笔记站，主要用于**自身学习、整理、加强记忆与反复巩固**。

## 项目定位

这个仓库当前更偏向个人使用，而不是对外维护的完整文档站。

- 用来记录自己在 `Java / 数据库 / 工业数字化 / AI` 等方向的学习过程
- 用输出来倒逼输入，把零散知识整理成可复习的内容
- 因为更新频率很高，所以 **README 暂时不放目录和详细规划**

这种边学边写、通过讲给自己看的方式加深理解，常被叫做：

- **费曼学习法**
- 也可以理解为 **输出倒逼输入**

## 本地开发

安装依赖：

```bash
npm install
```

启动开发：

```bash
npm run dev
```

构建：

```bash
npm run build
```
