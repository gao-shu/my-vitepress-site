工具推荐 ​

推荐一些全栈开发工程师日常工作中必需的工具及开源库。

集成开发环境（IDE） ​

IntelliJ IDEA（推荐） ​

  • 官网 - JetBrains IntelliJ IDEA
  • 下载页 - 社区版/旗舰版下载
  • 官方文档 - 使用文档
  • 特点: Java开发最强IDE，收费（有社区免费版）

PyCharm（Python开发） ​

  • 官网 - JetBrains PyCharm
  • 下载页 - 社区版/专业版下载
  • 特点: Python开发专用IDE

VS Code（全栈通用） ​

  • 官网 - Visual Studio Code
  • Java扩展 - Java语言支持
  • Python扩展 - Python语言支持
  • Vue扩展 - Vue.js支持
  • React扩展 - React/TypeScript支持
  • 特点: 轻量级，插件丰富，免费

WebStorm（前端开发） ​

  • 官网 - JetBrains WebStorm
  • 特点: 前端开发专用IDE

版本控制工具 ​

  • Git - 分布式版本控制
  • GitHub - 全球代码托管平台
  • Gitee - 国内代码托管平台
  • GitHub Desktop - GitHub可视化客户端
