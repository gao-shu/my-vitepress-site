# 关于我

这是一个基于 VitePress 构建的个人学习笔记站，主要用于**自身学习、整理、加强记忆与反复巩固**。

## 项目定位

这个站点当前更偏向个人使用，而不是对外维护的完整文档站。

- 用来记录自己在 `Java / 数据库 / 工业数字化 / AI / 英语口语` 等方向的学习过程
- 用输出来倒逼输入，把零散知识整理成可复习、可反复回看的内容
- 因为更新频率很高，所以这里暂时不做很重的对外介绍

这种边学边写、通过讲给自己看的方式加深理解，常被叫做：

- **费曼学习法**
- 也可以理解为 **输出倒逼输入**

## 当前原则

- 内容会频繁调整、增删、重组
- 现阶段以“自己能快速回看、快速补充”为第一优先级
- 若后续结构稳定，再逐步补更完整的说明

## 想保留的方向

- `Java`
- `数据库`
- `工业数字化`
- `AI`
- `英语口语`

## 说明

本站内容以个人学习整理为主，部分内容会持续迭代更新；如有不准确之处，后续会继续修正。
