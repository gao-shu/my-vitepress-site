# Node.js 开源项目

本页面精选了优秀的Node.js开源项目，包括Web框架、工具库、应用系统等。

## Web 框架

### 全栈框架
- **Express.js**：经典Web框架
  - GitHub: [https://github.com/expressjs/express](https://github.com/expressjs/express)
  - 特点: 轻量级、灵活、中间件机制
  - 适用: RESTful API、Web应用

- **NestJS**：企业级Node.js框架
  - GitHub: [https://github.com/nestjs/nest](https://github.com/nestjs/nest)
  - 特点: TypeScript、依赖注入、模块化
  - 适用: 企业应用、微服务

- **Koa.js**：下一代Web框架
  - GitHub: [https://github.com/koajs/koa](https://github.com/koajs/koa)
  - 特点: 异步处理、Context对象、轻量级
  - 适用: 现代Web应用

- **Fastify**：高性能Web框架
  - GitHub: [https://github.com/fastify/fastify](https://github.com/fastify/fastify)
  - 特点: 快速、JSON Schema验证、低开销
  - 适用: 高性能API

### 实时应用框架
- **Socket.IO**：实时双向通信
  - GitHub: [https://github.com/socketio/socket.io](https://github.com/socketio/socket.io)
  - 功能: WebSocket、实时通信、房间管理

- **Primus**：实时框架抽象层
  - GitHub: [https://github.com/primus/primus](https://github.com/primus/primus)
  - 功能: 多传输协议、自动降级

## 工具库

### HTTP客户端
- **Axios**：Promise-based HTTP客户端
  - GitHub: [https://github.com/axios/axios](https://github.com/axios/axios)
  - 功能: HTTP请求、拦截器、自动转换

- **Got**：人性化的HTTP请求库
  - GitHub: [https://github.com/sindresorhus/got](https://github.com/sindresorhus/got)
  - 功能: 简洁API、自动重试、流式响应

- **Node Fetch**：浏览器Fetch API实现
  - GitHub: [https://github.com/node-fetch/node-fetch](https://github.com/node-fetch/node-fetch)
  - 功能: 兼容浏览器Fetch API

### 工具库
- **Lodash**：实用工具函数库
  - GitHub: [https://github.com/lodash/lodash](https://github.com/lodash/lodash)
  - 功能: 函数式编程、数据操作

- **Moment.js**：日期处理库
  - GitHub: [https://github.com/moment/moment](https://github.com/moment/moment)
  - 功能: 日期解析、格式化、操作

- **UUID**：唯一标识符生成
  - GitHub: [https://github.com/uuidjs/uuid](https://github.com/uuidjs/uuid)
  - 功能: UUID生成、多种版本支持

### 异步处理
- **Async**：异步流程控制
  - GitHub: [https://github.com/caolan/async](https://github.com/caolan/async)
  - 功能: 并行/串行执行、流程控制

- **Bluebird**：Promise库
  - GitHub: [https://github.com/petkaantonov/bluebird](https://github.com/petkaantonov/bluebird)
  - 功能: Promise扩展、性能优化

## 数据库相关

### ORM
- **Sequelize**：多数据库ORM
  - GitHub: [https://github.com/sequelize/sequelize](https://github.com/sequelize/sequelize)
  - 支持: MySQL、PostgreSQL、SQLite等

- **TypeORM**：TypeScript ORM
  - GitHub: [https://github.com/typeorm/typeorm](https://github.com/typeorm/typeorm)
  - 特点: TypeScript支持、多数据库

- **Mongoose**：MongoDB ODM
  - GitHub: [https://github.com/Automattic/mongoose](https://github.com/Automattic/mongoose)
  - 功能: Schema定义、数据验证

### 数据库客户端
- **mysql2**：MySQL客户端
  - GitHub: [https://github.com/sidorares/node-mysql2](https://github.com/sidorares/node-mysql2)
  - 特点: 性能高、Promise支持

- **pg**：PostgreSQL客户端
  - GitHub: [https://github.com/brianc/node-postgres](https://github.com/brianc/node-postgres)
  - 功能: 连接池、查询构建

- **redis**：Redis客户端
  - GitHub: [https://github.com/redis/node-redis](https://github.com/redis/node-redis)
  - 功能: 集群支持、发布订阅
