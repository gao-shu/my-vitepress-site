# AI & 机器学习项目

本页面精选了AI、机器学习、深度学习相关的开源项目，包括框架、工具库、应用等。

## 深度学习框架

### TensorFlow 生态
- **TensorFlow**：Google开源机器学习框架
  - GitHub: [https://github.com/tensorflow/tensorflow](https://github.com/tensorflow/tensorflow)
  - 特点: 生产级、分布式训练、多种语言支持
  - 适用: 研究和生产环境

- **TensorFlow.js**：浏览器中运行的TensorFlow
  - GitHub: [https://github.com/tensorflow/tfjs](https://github.com/tensorflow/tfjs)
  - 功能: 浏览器ML、Node.js支持

- **TensorFlow Serving**：模型服务系统
  - GitHub: [https://github.com/tensorflow/serving](https://github.com/tensorflow/serving)
  - 功能: 模型部署、高性能推理

### PyTorch 生态
- **PyTorch**：Facebook开源深度学习框架
  - GitHub: [https://github.com/pytorch/pytorch](https://github.com/pytorch/pytorch)
  - 特点: 动态图、易用性好、研究友好
  - 适用: 学术研究、快速原型

- **PyTorch Lightning**：PyTorch轻量级包装
  - GitHub: [https://github.com/PyTorchLightning/pytorch-lightning](https://github.com/PyTorchLightning/pytorch-lightning)
  - 功能: 简化训练、自动优化

- **TorchServe**：PyTorch模型服务
  - GitHub: [https://github.com/pytorch/serve](https://github.com/pytorch/serve)
  - 功能: 模型部署、A/B测试

### 其他框架
- **JAX**：Google的数值计算库
  - GitHub: [https://github.com/google/jax](https://github.com/google/jax)
  - 特点: 可组合变换、自动微分

- **MXNet**：Apache深度学习框架
  - GitHub: [https://github.com/apache/mxnet](https://github.com/apache/mxnet)
  - 特点: 高效、灵活、可扩展
