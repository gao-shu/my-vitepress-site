# Vue.js 开源项目

本页面精选了优秀的Vue.js开源项目，包括UI框架、组件库、工具、应用等。

## 核心框架

- **Vue.js**：渐进式JavaScript框架
  - GitHub: [https://github.com/vuejs/vue](https://github.com/vuejs/vue)
  - 特点: 响应式数据绑定、组件化、轻量级

- **Vue 3**：最新版本
  - GitHub: [https://github.com/vuejs/core](https://github.com/vuejs/core)
  - 新特性: Composition API、更好的TypeScript支持

## UI 框架与组件库

### 桌面端组件库
- **Element Plus**：基于Vue 3的企业级组件库
  - GitHub: [https://github.com/element-plus/element-plus](https://github.com/element-plus/element-plus)
  - 特点: 完整组件、国际化、主题定制

- **Ant Design Vue**：Ant Design的Vue实现
  - GitHub: [https://github.com/vueComponent/ant-design-vue](https://github.com/vueComponent/ant-design-vue)
  - 特点: 企业级设计、美观大方

- **Vant**：轻量级移动端组件库
  - GitHub: [https://github.com/youzan/vant](https://github.com/youzan/vant)
  - 特点: 移动优先、按需加载

- **Quasar**：响应式Vue组件框架
  - GitHub: [https://github.com/quasarframework/quasar](https://github.com/quasarframework/quasar)
  - 特点: 跨平台、Material Design

### 移动端组件库
- **Mint UI**：Vue移动端组件库
  - GitHub: [https://github.com/ElemeFE/mint-ui](https://github.com/ElemeFE/mint-ui)
  - 特点: 饿了么出品、移动端优化

- **Cube UI**：滴滴移动端组件库
  - GitHub: [https://github.com/didi/cube-ui](https://github.com/didi/cube-ui)
  - 特点: 丰富组件、性能优化

## 路由与状态管理

### 路由
- **Vue Router**：官方路由管理器
  - GitHub: [https://github.com/vuejs/vue-router](https://github.com/vuejs/vue-router)
  - 功能: 路由映射、嵌套路由、路由守卫

- **Vue Router 4**：Vue 3版本
  - GitHub: [https://github.com/vuejs/router](https://github.com/vuejs/router)
  - 新特性: TypeScript支持、组合式API

### 状态管理
- **Pinia**：Vue的状态管理库
  - GitHub: [https://github.com/vuejs/pinia](https://github.com/vuejs/pinia)
  - 特点: TypeScript友好、直观API

- **Vuex**：官方状态管理库
  - GitHub: [https://github.com/vuejs/vuex](https://github.com/vuejs/vuex)
  - 功能: 集中式状态管理、插件系统

- **Vuex 4**：Vue 3兼容版本
  - GitHub: [https://github.com/vuejs/vuex](https://github.com/vuejs/vuex)
  - 特点: 更好的TypeScript支持
