# 物联网平台

本页面精选了物联网相关的开源项目，帮助中小企业构建物联网应用。

## 推荐项目

- **ThingsBoard**：开源物联网平台，支持设备管理、数据可视化。
  - GitHub: [https://github.com/thingsboard/thingsboard](https://github.com/thingsboard/thingsboard)

- **OpenIoT**：轻量级物联网框架。
  - GitHub: [https://github.com/OpenIoT/OpenIoT](https://github.com/OpenIoT/OpenIoT)

这些项目适用于设备连接、数据采集和监控等物联网场景。
