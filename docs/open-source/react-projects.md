# React 开源项目

本页面精选了优秀的React开源项目，包括框架、组件库、工具、应用等。

## 核心框架

- **React**：用于构建用户界面的JavaScript库
  - GitHub: [https://github.com/facebook/react](https://github.com/facebook/react)
  - 特点: 组件化、声明式、虚拟DOM

- **React 18**：最新稳定版本
  - 新特性: 并发渲染、Suspense、自动批处理

## UI 框架与组件库

### 桌面端组件库
- **Material-UI (MUI)**：Material Design组件库
  - GitHub: [https://github.com/mui/material-ui](https://github.com/mui/material-ui)
  - 特点: Material Design、主题定制、国际化

- **Ant Design**：企业级UI设计语言
  - GitHub: [https://github.com/ant-design/ant-design](https://github.com/ant-design/ant-design)
  - 特点: 企业级、完整组件、设计规范

- **Chakra UI**：简单灵活的组件库
  - GitHub: [https://github.com/chakra-ui/chakra-ui](https://github.com/chakra-ui/chakra-ui)
  - 特点: 组合式API、主题定制、暗色模式

- **React Bootstrap**：Bootstrap组件的React实现
  - GitHub: [https://github.com/react-bootstrap/react-bootstrap](https://github.com/react-bootstrap/react-bootstrap)
  - 特点: Bootstrap兼容、响应式设计

### 移动端组件库
- **React Native Elements**：跨平台UI工具包
  - GitHub: [https://github.com/react-native-elements/react-native-elements](https://github.com/react-native-elements/react-native-elements)
  - 功能: 丰富的组件、主题支持

- **NativeBase**：移动端组件库
  - GitHub: [https://github.com/GeekyAnts/NativeBase](https://github.com/GeekyAnts/NativeBase)
  - 特点: 通用组件、平台适配

## 路由与状态管理

### 路由
- **React Router**：声明式路由
  - GitHub: [https://github.com/remix-run/react-router](https://github.com/remix-run/react-router)
  - 功能: 客户端路由、代码分割、嵌套路由

- **React Router v6**：最新版本
  - 新特性: 更简单的API、更好的TypeScript支持

### 状态管理
- **Redux**：可预测的状态容器
  - GitHub: [https://github.com/reduxjs/redux](https://github.com/reduxjs/redux)
  - 特点: 单向数据流、中间件、时间旅行调试

- **Redux Toolkit**：Redux官方推荐工具包
  - GitHub: [https://github.com/reduxjs/redux-toolkit](https://github.com/reduxjs/redux-toolkit)
  - 功能: 简化Redux、减少样板代码

- **Zustand**：轻量级状态管理
  - GitHub: [https://github.com/pmndrs/zustand](https://github.com/pmndrs/zustand)
  - 特点: 简单API、TypeScript友好

- **Recoil**：React状态管理库
  - GitHub: [https://github.com/facebookexperimental/Recoil](https://github.com/facebookexperimental/Recoil)
  - 功能: 原子化状态、派生状态
