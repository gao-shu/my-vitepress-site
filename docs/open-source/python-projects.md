# Python 开源项目

本页面精选了优秀的Python开源项目，包括Web框架、数据科学、机器学习、自动化工具等。

## Web 框架

### 全栈框架
- **Django**：高水平全栈Web框架
  - GitHub: [https://github.com/django/django](https://github.com/django/django)
  - 特点: MTV架构、自带Admin、ORM、安全性高
  - 适用: 企业级应用、CMS系统

- **Flask**：轻量级Web框架
  - GitHub: [https://github.com/pallets/flask](https://github.com/pallets/flask)
  - 特点: 微框架、扩展性强、简单灵活
  - 适用: API开发、小型应用

- **FastAPI**：现代异步Web框架
  - GitHub: [https://github.com/tiangolo/fastapi](https://github.com/tiangolo/fastapi)
  - 特点: 异步支持、自动API文档、类型提示
  - 适用: 高性能API、微服务

### 异步框架
- **Tornado**：异步Web框架
  - GitHub: [https://github.com/tornadoweb/tornado](https://github.com/tornadoweb/tornado)
  - 特点: 非阻塞I/O、高并发处理
  - 适用: 实时应用、WebSocket

- **Sanic**：异步Web框架
  - GitHub: [https://github.com/sanic-org/sanic](https://github.com/sanic-org/sanic)
  - 特点: 基于uvloop、性能极高
  - 适用: 高性能Web应用

## 数据科学与机器学习

### 数据处理
- **pandas**：数据分析和处理库
  - GitHub: [https://github.com/pandas-dev/pandas](https://github.com/pandas-dev/pandas)
  - 功能: DataFrame操作、数据清洗、时间序列

- **NumPy**：数值计算库
  - GitHub: [https://github.com/numpy/numpy](https://github.com/numpy/numpy)
  - 功能: 多维数组、数学函数、线性代数

- **SciPy**：科学计算库
  - GitHub: [https://github.com/scipy/scipy](https://github.com/scipy/scipy)
  - 功能: 优化、积分、信号处理

### 机器学习
- **scikit-learn**：经典机器学习库
  - GitHub: [https://github.com/scikit-learn/scikit-learn](https://github.com/scikit-learn/scikit-learn)
  - 功能: 分类、回归、聚类、降维

- **TensorFlow**：深度学习框架
  - GitHub: [https://github.com/tensorflow/tensorflow](https://github.com/tensorflow/tensorflow)
  - 特点: 生产级、分布式训练、移动端支持

- **PyTorch**：深度学习框架
  - GitHub: [https://github.com/pytorch/pytorch](https://github.com/pytorch/pytorch)
  - 特点: 动态图、易用性好、研究友好

### 数据可视化
- **Matplotlib**：绘图库
  - GitHub: [https://github.com/matplotlib/matplotlib](https://github.com/matplotlib/matplotlib)
  - 功能: 2D/3D绘图、统计图表

- **Seaborn**：统计数据可视化
  - GitHub: [https://github.com/mwaskom/seaborn](https://github.com/mwaskom/seaborn)
  - 功能: 统计图表、美观样式

- **Plotly**：交互式图表
  - GitHub: [https://github.com/plotly/plotly.py](https://github.com/plotly/plotly.py)
  - 功能: Web交互图表、仪表板
