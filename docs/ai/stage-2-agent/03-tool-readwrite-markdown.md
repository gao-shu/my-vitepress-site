# 03 做一个工具：读写 Markdown（本地知识库）

> 目标：先把 Tools 做出来，再接 Agent。你只要实现“读/写 Markdown”两个能力就够了。

## 1）工具清单（最小两件事）

- **read**：给一个路径，读出 Markdown 文本
- **write**：给一个路径与内容，写入 Markdown 文件

## 2）建议约束（避免写坏你的知识库）

- 只允许写到指定目录（例如 `generated/`）
- 写文件前做一次存在性检查（避免覆盖）
- 每次写入都记录日志：时间、路径、字数

## 3）今日练习

- 手写 `read_markdown()` / `write_markdown()` 两个函数
- 用它们做一次“读 → 写”的复制动作（不接 LLM）
