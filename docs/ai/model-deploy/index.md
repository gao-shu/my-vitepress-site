## 你学完应该能做到

- 能在本地把一个可用模型跑起来（含下载、推理、参数设置、基本排障）
- 能在云端用 API 快速接入，或用 GPU 实例自建推理服务
- 能按“环境/场景”选模型，不盲目追新追大
