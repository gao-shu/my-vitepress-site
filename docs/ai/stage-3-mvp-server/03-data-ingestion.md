# 03 数据接入：扫描 docs/ + 统一结构

> 目标：先接入你最容易获得的数据：本地 `docs/`。跑通“扫描 → 抽取 → 输出”闭环。

## 1）接入策略（最简单可行）

- 遍历 `docs/` 下的 `.md`
- 抽取：
  - 一级标题（作为 `title`）
  - 若干小节标题（作为 `outline`）
  - 前 200～500 字（作为 `snippet`）

## 2）更新策略（先别做复杂定时）

第一版建议：

- 服务启动时扫描一次
- 或提供一个 `POST /reindex` 手动触发重建索引

## 3）验收标准

- `/modules` 能看到你现有模块
- `/context?module=database` 能返回数据库模块的一批片段
