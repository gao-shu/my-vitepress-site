# 阶段三 Checklist：合格线

## 1）服务（必须）

- 你有一个可运行的服务项目（任意语言栈）
- 有至少 2 个接口：
  - `/modules`
  - `/context?module=xxx`

## 2）数据（必须）

- 至少接入 1 个数据源（建议先接本地 `docs/`）
- 做了统一结构输出（字段稳定）

## 3）串联（必须）

- 你的 Agent 已经改成：
  - 先调中台拿 context
  - 再调 LLM 生成内容

做到这些，前三阶段就形成一个完整闭环：

**LLM Prompt（阶段一） → Tools/Agent（阶段二） → 数据中台（阶段三）**。
