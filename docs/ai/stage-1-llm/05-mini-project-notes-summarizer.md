# 05 小项目：面试笔记总结器（读取 docs/ 下 Markdown）

> 目标：做出一个你“马上用得上”的小工具：把某篇笔记总结成固定模板，并输出为新的 Markdown 文件。

## 1）功能定义（不要贪多）

- 输入：一个 Markdown 文件路径（比如你 VitePress 的某篇文档）
- 输出：一个新的 Markdown（总结版），包含：
  - 5 点要点
  - 3 个面试追问
  - 3 个易错点
  - 1 段“口语化回答模板”

## 2）实现步骤

### 第一步：读取文件内容

- 用 Python 读取 Markdown 文本（注意编码 UTF-8）

### 第二步：拼 Prompt（固定模板）

- system：固定输出模板、口吻、边界（不编造、只基于输入）
- user：贴入原文内容

### 第三步：调用 LLM API

- 控制输出为 Markdown

### 第四步：落盘到 outputs/

- 文件名可以按原文件名 + `-summary.md`

## 3）验收标准（做到就算过关）

- 你可以对 `docs/resume/skills-interview-prep.md` 的某一小节做总结
- 生成结果是可读、可直接复制到面试回答里的
