# 阶段一 Checklist：合格线（你做到这些就可以进入下一阶段）

## 1）Prompt（必须）

- 你有 **3 套可复用 Prompt**：
  - 面试题生成（Q/A/追问/易错点）
  - 学习总结（要点/例子/坑）
  - 简历优化（动作/技术/结果）
- 你能用“输出格式强约束”让结果稳定（固定 Markdown 栏目）。

## 2）API（建议）

- 你能用 Python 跑通一次模型 API 调用：
  - 环境变量存 Key
  - 从文件读取输入
  - 输出写到 `outputs/*.md`

## 3）最小项目（必须）

- 你完成了“面试笔记总结器”：
  - 随机指定一篇 Markdown → 自动输出总结版 Markdown

做到这些，你就可以进入 **阶段二：Agent（让模型调用工具干活）**。
