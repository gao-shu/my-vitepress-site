- 能稳定在网页端用 1～2 个模型做：总结、改写、生成面试 Q&A、解释报错。
- 能写出 3 套可复用 Prompt：**面试题生成 / 学习总结 / 简历优化**。
- 能用 Python 写一个脚本：读入文本 → 调用 LLM API → 输出 Markdown 文件。
