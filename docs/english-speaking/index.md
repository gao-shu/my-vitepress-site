# 英语口语

> 这个模块主要用于 **英语口语学习、反复练习、场景表达积累**。  
> 目标不是研究语法体系，而是：**先敢开口，再说自然，再逐步熟练**。

---

## 模块定位

本模块主要做三件事：

1. **打基础**：先把 `48 音标` 和常见发音问题弄清楚  
2. **背模板**：按场景积累常用句型，减少“想说但说不出来”的卡顿  
3. **做输出**：围绕日常、工作、会议、电话、社交等场景，反复练习

## 怎么使用这个模块

- **不要贪多**：每天只记 10-20 句
- **先照着读**：先模仿，不急着自己造句
- **场景化复习**：比如今天只练“开会表达”，明天只练“日常寒暄”
- **反复替换词汇**：同一句模板，用不同人名、时间、地点替换

---

## 学习原则

- 先追求 **能说出来**
- 再追求 **说得自然**
- 最后再追求 **更地道、更丰富**

> 口语提升的关键不是一次背很多，而是**高频重复 + 场景复现**。
