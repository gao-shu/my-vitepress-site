# 安全技术栈

网络安全和信息安全是现代IT系统的基石，保护数据、系统和用户免受威胁。本章介绍安全架构、认证授权、加密技术、网络安全和安全运维。

## 安全架构设计

### 零信任架构
- **核心原则**: 永不信任、始终验证、最小权限
- **关键组件**: 身份验证、设备验证、访问控制、持续监控
- **实施策略**: 微分段、加密传输、API安全、用户行为分析
- **技术栈**: BeyondCorp、Google Zero Trust、Microsoft Zero Trust
- **最佳实践**: 渐进式实施、自动化验证、持续评估

### 安全信息与事件管理 (SIEM)
- **核心功能**: 日志聚合、实时分析、威胁检测、事件响应
- **主流产品**: Splunk、ELK Stack、IBM QRadar、RSA NetWitness
- **开源方案**: Wazuh、OSSIM、Prelude SIEM
- **分析技术**: 规则引擎、机器学习、行为分析、关联分析
- **最佳实践**: 日志标准化、告警调优、事件关联

### 安全编排自动化响应 (SOAR)
- **核心能力**: 威胁情报整合、自动化响应、案例管理
- **主流平台**: IBM Resilient、Splunk Phantom、Palo Alto Cortex XSOAR
- **开源方案**: Shuffle SOAR、TheHive、OpenCTI
- **工作流**: 事件触发、情报 enrichment、响应执行、效果评估
- **最佳实践**: 剧本开发、集成配置、持续优化

## 身份与访问管理

### 身份认证 (Authentication)
- **传统认证**: 用户名密码、证书认证、智能卡
- **多因素认证**: 知识因素、所有物因素、生物识别
- **现代认证**: OAuth 2.0、OpenID Connect、SAML
- **无密码认证**: FIDO2/WebAuthn、生物识别、设备认证
- **最佳实践**: 密码策略、会话管理、认证日志

### 访问控制 (Authorization)
- **访问控制模型**: DAC (自主访问控制)、MAC (强制访问控制)、RBAC (基于角色的访问控制)
- **属性-based访问控制**: ABAC、XACML标准、动态授权
- **权限管理**: 最小权限原则、权限继承、权限审查
- **技术实现**: LDAP/AD、OAuth scopes、JWT claims
- **最佳实践**: 权限分级、审计记录、定期审查

### 身份管理平台
- **目录服务**: Active Directory、OpenLDAP、FreeIPA
- **IAM平台**: Okta、Auth0、Keycloak、Azure AD
- **用户生命周期**: 账户创建、权限分配、账户维护、账户注销
- **自助服务**: 密码重置、访问请求、权限管理
- **最佳实践**: 目录同步、SSO配置、合规审计

## 加密技术

### 对称加密
- **经典算法**: AES (高级加密标准)、DES、3DES、RC4
- **块加密模式**: ECB、CBC、CFB、OFB、CTR
- **密钥管理**: 密钥生成、密钥分发、密钥轮换、密钥销毁
- **应用场景**: 数据加密、文件加密、数据库加密
- **最佳实践**: 密钥长度选择、模式选择、安全实现

### 非对称加密
- **经典算法**: RSA、DSA、ECDSA、Ed25519
- **密钥交换**: Diffie-Hellman、ECDH
- **数字签名**: 签名生成、签名验证、证书链验证
- **应用场景**: SSL/TLS、数字证书、密钥交换
- **最佳实践**: 密钥长度选择、算法选择、安全参数

### 哈希函数
- **密码学哈希**: SHA-256、SHA-3、Blake2、Argon2
- **HMAC**: 带密钥的哈希消息认证码
- **应用场景**: 密码存储、数据完整性、数字签名
- **最佳实践**: 盐值使用、工作因子调整、抗碰撞性

### 密钥管理
- **密钥生命周期**: 生成、分发、使用、轮换、销毁
- **密钥存储**: HSM (硬件安全模块)、KMS (密钥管理服务)
- **密钥分发**: 密钥协商、密钥封装、公钥基础设施
- **合规要求**: FIPS 140-2、PCI DSS、数据主权
- **最佳实践**: 密钥备份、访问控制、审计日志

## 网络安全

### 网络边界安全
- **防火墙**: 包过滤防火墙、状态检测防火墙、下一代防火墙
- **入侵检测**: Snort、Suricata、Zeek (Bro)
- **入侵防御**: IPS系统、蜜罐技术、欺骗技术
- **最佳实践**: 规则优化、误报处理、性能调优

### 网络分段
- **VLAN分段**: 广播域隔离、二层安全
- **子网分段**: 三层隔离、路由控制
- **微分段**: 软件定义分段、零信任网络
- **技术实现**: SDN (软件定义网络)、NFV (网络功能虚拟化)
- **最佳实践**: 分段策略、流量控制、监控覆盖

### VPN与远程访问
- **IPsec VPN**: 站点到站点、远程访问、传输模式
- **SSL VPN**: 客户端less访问、应用级访问
- **WireGuard**: 现代VPN协议、高性能、简单配置
- **最佳实践**: 证书管理、双因素认证、访问控制

### Web应用安全
- **OWASP Top 10**: 注入攻击、认证失效、敏感数据泄露等
- **WAF**: ModSecurity、Cloudflare WAF、AWS WAF
- **API安全**: OAuth 2.0、JWT、API密钥管理
- **最佳实践**: 输入验证、安全编码、安全测试

## 应用安全

### 安全开发生命周期 (SDLC)
- **安全需求**: 威胁建模、风险评估、安全需求定义
- **安全设计**: 安全架构、安全模式、威胁缓解
- **安全编码**: 安全编码规范、代码审查、静态分析
- **安全测试**: SAST、DAST、IAST、渗透测试
- **最佳实践**: DevSecOps、自动化安全、持续监控

### 容器安全
- **镜像安全**: 漏洞扫描、基线镜像、最小化镜像
- **运行时安全**: 容器隔离、资源限制、监控日志
- **编排安全**: Kubernetes安全、RBAC、网络策略
- **工具**: Docker Security Scanning、Aqua Security、Twistlock
- **最佳实践**: 镜像签名、运行时保护、合规检查

### 云安全
- **云安全模型**: 责任共担模型、云访问安全代理 (CASB)
- **云安全工具**: AWS Security Hub、Azure Security Center、GCP Security Command Center
- **配置管理**: CloudFormation安全、Terraform安全、基础设施即代码安全
- **最佳实践**: 最小权限、加密存储、多云安全

## 数据安全

### 数据保护
- **数据分类**: 公开数据、内部数据、敏感数据、机密数据
- **数据加密**: 静态加密、传输加密、使用中加密
- **数据脱敏**: 掩码、替换、泛化、差分隐私
- **最佳实践**: 分类标签、加密策略、访问控制

### 数据丢失防护 (DLP)
- **DLP策略**: 内容识别、上下文分析、用户行为
- **主流产品**: Symantec DLP、McAfee DLP、Microsoft DLP
- **开源方案**: OpenDLP、MyDLP
- **部署模式**: 端点DLP、网络DLP、云DLP
- **最佳实践**: 策略调优、误报处理、用户培训

### 数据库安全
- **访问控制**: 数据库用户、角色权限、行级安全
- **审计功能**: 语句审计、对象审计、权限审计
- **加密功能**: 透明数据加密 (TDE)、列级加密
- **最佳实践**: 最小权限、审计启用、安全配置

## 安全运维

### 漏洞管理
- **漏洞扫描**: Nessus、OpenVAS、Qualys、Rapid7
- **漏洞评估**: CVSS评分、 exploitability、影响评估
- **补丁管理**: 补丁识别、测试部署、回滚策略
- **最佳实践**: 扫描频率、补丁优先级、变更管理

### 事件响应
- **事件响应流程**: 识别、遏制、根除、恢复、经验总结
- **事件响应团队**: CERT、CSIRT、SOC (安全运营中心)
- **工具支持**: 事件管理系统、案例管理、知识库
- **最佳实践**: 响应时间、沟通协调、持续改进

### 安全监控
- **安全监控**: IDS/IPS、SIEM、EDR (端点检测响应)
- **威胁情报**: 外部威胁情报、内部威胁检测
- **合规监控**: 安全策略合规、审计要求满足
- **最佳实践**: 监控覆盖、告警调优、报告生成

### 取证分析
- **数字取证**: 证据收集、证据保存、证据分析
- **内存取证**: Volatility框架、内存镜像分析
- **日志取证**: 日志聚合、时间线分析、关联分析
- **最佳实践**: 证据链维护、工具验证、无污染分析

## 合规与审计

### 安全标准
- **国际标准**: ISO 27001、NIST框架、PCI DSS
- **行业标准**: HIPAA (医疗)、SOX (财务)、GDPR (隐私)
- **国家标准**: 等保2.0、等级保护制度
- **最佳实践**: 差距分析、合规实施、持续审计

### 审计与报告
- **安全审计**: 内部审计、外部审计、渗透测试
- **审计日志**: 系统日志、安全事件、用户活动
- **合规报告**: 审计报告、合规证明、风险评估
- **最佳实践**: 日志完整性、审计覆盖、报告自动化

### 风险管理
- **风险评估**: 资产识别、威胁识别、脆弱性评估
- **风险量化**: 风险评分、影响评估、概率计算
- **风险缓解**: 控制措施、转移策略、接受策略
- **最佳实践**: 风险矩阵、缓解计划、监控评估
