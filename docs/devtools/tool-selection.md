# 工具选型建议

> 所属专题：[开发工具与环境配置](/devtools/)

### 11.1 新手入门推荐组合

```
💰 免费组合：
- JDK：Adoptium 开源版本
- IDE：IntelliJ IDEA Community
- 数据库：DBeaver
- API 测试：Apifox
- Git：原生命令行 + GitKraken（免费）
- 文本编辑：VS Code
```

### 11.2 企业级推荐组合

```
💼 专业组合：
- JDK：Oracle JDK（商业授权）
- IDE：IntelliJ IDEA Ultimate
- 数据库：Navicat Premium / DataGrip
- API 测试：Postman Team
- Git：GitLab Enterprise
- 容器：Docker Desktop Business
```

### 11.3 极简主义组合

```
🎯 轻量组合：
- JDK：jEnv（多版本管理）
- IDE：VS Code + Java 扩展包
- 数据库：命令行客户端
- API 测试：curl / httpie
- Git：命令行
```
