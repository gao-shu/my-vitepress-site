# 数据库客户端工具

> 所属专题：[开发工具与环境配置](/devtools/)

### 6.1 Navicat（功能强大，收费）

**特点：**
- ✅ 支持多种数据库（MySQL、PostgreSQL、Oracle 等）
- ✅ 可视化界面，操作简单
- ✅ 数据同步、备份功能完善
- ❌ 价格较贵（约 $999/年）

**常用功能：**
- 数据库连接管理
- SQL 查询编辑器
- 数据传输与同步
- 模型设计器
- 报表生成

### 6.2 DataGrip（JetBrains 出品，收费）

**特点：**
- ✅ 智能 SQL 补全
- ✅ 强大的重构功能
- ✅ 与 IDEA 无缝集成
- ✅ 支持 20+ 种数据库
- ❌ 需要单独购买（约 $199/年）

**实用技巧：**
```sql
-- 使用快捷键快速执行
Ctrl + Enter  -- 执行 SQL
Ctrl + Q      -- 快速文档
Ctrl + B      -- 跳转到表定义
```

### 6.3 DBeaver（免费开源，强烈推荐）⭐

**下载地址：**
```
https://dbeaver.io/download/
```

**特点：**
- ✅ 完全免费，开源
- ✅ 支持几乎所有主流数据库
- ✅ 功能全面，不输商业软件
- ✅ 跨平台（Windows/Mac/Linux）

**核心功能：**
- ER 图可视化
- 数据导出导入
- SQL 编辑器
- 驱动自动下载
- 元数据浏览器
