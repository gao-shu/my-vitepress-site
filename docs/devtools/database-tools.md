# 数据库客户端工具

> 所属专题：[开发工具与环境配置](/devtools/)

## 1) Navicat

- 下载地址：<https://www.navicat.com/en/download/navicat-premium-lite>
- 核心功能：
  - 数据库连接管理
  - SQL 编辑与执行
  - 表结构与数据可视化查看
- 简单配置：
  - 新建连接：`Connection -> MySQL`
  - 填写 `Host` / `Port(3306)` / `Username` / `Password`
  - 点击 `Test Connection` 成功后保存

## 2) DBeaver

- 下载地址：<https://dbeaver.io/download/>
- 核心功能：
  - 多数据库连接管理
  - SQL 编辑与执行
  - 数据导入导出
- 简单配置：
  - 新建连接：`Database -> New Database Connection`
  - 选择 `MySQL`，填写 `Host` / `Port(3306)` / `Database` / `Username` / `Password`
  - 首次连接按提示下载驱动，完成后点击 `Test Connection`
