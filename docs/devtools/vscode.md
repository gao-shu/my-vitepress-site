# VS Code 安装与配置

> 所属专题：[开发工具与环境配置](/devtools/)

## 1. 下载与安装

- 下载地址：<https://code.visualstudio.com/>

安装完成后，建议勾选：

- `Add to PATH`
- `Open with Code`

## 2. 常用扩展

- `Extension Pack for Java`
- `Python`
- `Volar`
- `Docker`
- `GitLens`

## 3. 简单配置建议

- 自动保存：`File -> Auto Save`
- 字体大小：`Settings -> Editor: Font Size`
- 终端默认 Shell：`Settings -> Terminal > Default Profile`

## 4. 适合场景

- 轻量开发
- 前端 / Node.js
- 日常脚本与文档编辑
