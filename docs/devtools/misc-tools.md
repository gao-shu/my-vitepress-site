# 其他实用工具

> 所属专题：[开发工具与环境配置](/devtools/)

### 8.1 终端工具

**Windows Terminal（Win10/11 必备）**
```bash
# Microsoft Store 下载安装
# 支持多标签、分屏、自定义主题
```

**Oh My Zsh（Mac/Linux）**
```bash
# 安装
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

# 推荐插件
plugins=(git docker mvn gradle)
```

### 8.2 文本编辑器

**VS Code（轻量级首选）**
```bash
# 下载地址
https://code.visualstudio.com/

# 推荐扩展
- Extension Pack for Java
- Spring Boot Extension Pack
- Vue Language Features
- Docker
- GitLens
```

### 8.3 截图工具

**Snipaste（强烈推荐）**
```bash
# 下载地址
https://www.snipaste.com/

# 特点
- F1 截图，F3 贴图
- 自动检测 UI 元素
- 标注功能完善
```

### 8.4 文件搜索

**Everything（Windows 神器）**
```bash
# 下载地址
https://www.voidtools.com/

# 特点
- 秒搜全盘文件
- 支持正则表达式
- 体积小巧
```
